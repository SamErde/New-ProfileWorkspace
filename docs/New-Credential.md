# New-Credential

## SYNOPSIS
Create a new secure credential.

## SYNTAX

```
New-Credential [<CommonParameters>]
```

## DESCRIPTION
Create a new secure credential to use in other functions.

## EXAMPLES

### EXAMPLE 1
```
$Credential = New-Credential
```
