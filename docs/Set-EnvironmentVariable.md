# Set-EnvironmentVariable

## SYNOPSIS
Set environment variables.

## SYNTAX

```
Set-EnvironmentVariable [-Name] <String> [-Value] <String> [[-Target] <EnvironmentVariableTarget>]
 [<CommonParameters>]
```

## DESCRIPTION
